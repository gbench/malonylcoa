> 词汇：集合
> 语法：运算
> 空间：词汇 + 语法 的 代数结构！
> 代数结构： 一套 对 （数学）概念 进行 相互组织 & 连接 的 功能体系&语法系统，旨在对 数学思维（构造&分解 目标价值意义的形式逻辑） 进行 表达&论证。
> 赋范空间：可构造出 长度概念 概念的基础性 代数结构，赋范空间 旨在 为概念&结构提供出一种确定的 数值量量感 。
> 度量空间：明确指定概念&结构为二元关系（X，Y） 形式的 赋范空间。
> 序列结构：一种可以表打先后次序关系的代数结构，基础形式为 Seq={a_1,a_2, ..., a_n, ...}
> 柯西序列：一种可以在序列结构上构建收敛性的代数结构。 强调关系长度逐级递减:  length (a_{n-1}, a_{n})  > length (a_{n-1}, a_{n})
> 完备空间： 可令柯西序列收敛的度量空间。
> 内积空间：可表达 几何思维 （角度+长度）代数结构。

### 2. 代数结构接口

#### 2.1 群 (Group) = 可逆运算

```haskell
-- 半群 (Semigroup): 只有结合律
class Semigroup a where
    (<>) :: a -> a -> a
    -- 定律: (x <> y) <> z = x <> (y <> z)

-- 幺半群 (Monoid): 半群 + 单位元
class Semigroup a => Monoid a where
    mempty :: a
    -- 定律: mempty <> x = x, x <> mempty = x

-- 群 (Group): 幺半群 + 逆元
class Monoid a => Group a where
    inverse :: a -> a
    -- 定律: x <> inverse x = mempty, inverse x <> x = mempty

-- 示例：整数加法群
instance Group Integer where
    inverse = negate
    -- 继承自 Monoid Integer 的 mempty = 0, (<>) = (+)
```

---

#### 2.2 赋范空间 (Normed Space) = 可测长度的线性空间

```haskell
-- 向量空间（简化版，只取实数域）
class VectorSpace v where
    zero   :: v
    (^+^)  :: v -> v -> v          -- 向量加法
    (*^)   :: Double -> v -> v      -- 标量乘法
    -- 定律：向量空间公理（结合律、分配律、单位元等）

-- 赋范空间：可测长度的向量空间
class VectorSpace v => NormedSpace v where
    norm :: v -> Double
    -- 定律: 
    --   norm (x ^+^ y) <= norm x + norm y   (三角不等式)
    --   norm (c *^ x) == abs c * norm x     (齐次性)
    --   norm x >= 0, 且 norm x == 0 当且仅当 x == zero

-- 示例：二维欧氏向量
data V2 = V2 Double Double deriving (Show, Eq)

instance VectorSpace V2 where
    zero = V2 0 0
    (V2 x1 y1) ^+^ (V2 x2 y2) = V2 (x1+x2) (y1+y2)
    c *^ (V2 x y) = V2 (c*x) (c*y)

instance NormedSpace V2 where
    norm (V2 x y) = sqrt (x*x + y*y)
```

#### 2.3 度量空间 (Metric Space) = 可测距离的类型

```haskell
class MetricSpace a where
    distance :: a -> a -> Double
    -- 定律:
    --   distance x y >= 0
    --   distance x y == 0 当且仅当 x == y
    --   distance x y == distance y x          (对称性)
    --   distance x z <= distance x y + distance y z  (三角不等式)

-- 任何赋范空间都可以导出度量（自动实例）
instance NormedSpace a => MetricSpace a where
    distance x y = norm (x ^+^ (negateV y))  -- 需要 negateV
  where
    negateV v = (-1) *^ v  -- 向量取负

-- 示例：字符串的编辑距离（Levenshtein distance）
instance MetricSpace String where
    distance = levenshteinDist  -- 具体实现略
```

#### 2.4 序列结构 (Sequence) = 可迭代的流

```haskell
-- 方案1：列表（有限）
type Seq a = [a]

-- 方案2：无限流（惰性列表）
type Stream a = [a]  -- 惰性求值天然支持无限

-- 方案3：更通用的可迭代抽象
class Sequence s where
    type Element s :: Type
    next :: s -> Maybe (Element s, s)  -- 类似 uncons
    -- 可以表示有限或无限

-- 示例：列表自然就是序列
instance Sequence [a] where
    type Element [a] = a
    next [] = Nothing
    next (x:xs) = Just (x, xs)
```

---

#### 2.5 柯西序列 (Cauchy Sequence) = 带稳定化检查的序列

```haskell
-- 一个柯西序列是一个序列，加上一个“稳定化函数”
data CauchySequence a = CauchySeq
    { seqGen   :: Int -> a           -- 生成第 n 个元素
    , stabilize :: Double -> Int     -- 给定 epsilon，返回 N，使得 m,n>N 时距离 < epsilon
    }

-- 检查一个序列是否是柯西序列（测试函数）
isCauchy :: MetricSpace a => CauchySequence a -> Bool
isCauchy (CauchySeq gen stabilize) = 
    all (\eps -> let n = stabilize eps
                 in  all (\m -> all (\k -> distance (gen m) (gen k) < eps) 
                                [n+1 .. n+10])  -- 抽样检查
        ) [1, 0.1, 0.01, 0.001]  -- 多精度测试

-- 收敛的数列构成柯西序列（例如 1/n）
cauchy_1_over_n :: CauchySequence Double
cauchy_1_over_n = CauchySeq 
    { seqGen = \n -> 1 / fromIntegral n
    , stabilize = \eps -> ceiling (1 / eps) + 1
    }
```

#### 2.6 完备空间 (Complete Space) = 可求极限的空间

```haskell
class MetricSpace a => CompleteSpace a where
    -- 任何柯西序列都能收敛到该空间中的一个点
    limit :: CauchySequence a -> a
    -- 定律: for any epsilon > 0, 
    --       distance (limit seq) (seqGen seq (stabilize seq epsilon)) < epsilon

-- 示例：实数空间是完备的
instance CompleteSpace Double where
    limit (CauchySeq gen _) = -- 用数值方法逼近极限
        let eps = 1e-10
            n = round (logBase 2 (1/eps)) + 10
        in gen n  -- 取足够后面的项作为近似

-- 示例：有理数空间不完备（不实现 CompleteSpace）
-- 因为 sqrt(2) 的柯西序列在有理数中不收敛
```

#### 2.7 内积空间 (Inner Product Space) = 有角度和长度的空间

```haskell
class VectorSpace v => InnerProductSpace v where
    dot :: v -> v -> Double
    -- 定律:
    --   dot x x >= 0
    --   dot x x == 0 当且仅当 x == zero
    --   dot x y == dot y x                     (对称性)
    --   dot (c *^ x) y == c * dot x y          (线性性)
    --   dot (x ^+^ y) z == dot x z + dot y z   (分配律)

    -- 可由内积导出的范数（自动实现）
    normFromInner :: v -> Double
    normFromInner x = sqrt (dot x x)

    -- 角度（自动实现）
    angle :: v -> v -> Double
    angle x y = acos (dot x y / (normFromInner x * normFromInner y))

-- 示例：V2 的内积空间实例
instance InnerProductSpace V2 where
    dot (V2 x1 y1) (V2 x2 y2) = x1*x2 + y1*y2
    -- normFromInner 自动继承

-- 使用角度
v1 = V2 1 0
v2 = V2 0 1
-- angle v1 v2 = pi/2 (90度)
```

### 3. 核心结构之间的函数关系（Haskell 表达）

```haskell
-- 1. 内积空间 → 赋范空间（自动派生）
instance InnerProductSpace v => NormedSpace v where
    norm = normFromInner

-- 2. 赋范空间 → 度量空间（自动派生）
instance NormedSpace v => MetricSpace v where
    distance x y = norm (x ^+^ negateV y)
  where
    negateV v = (-1) *^ v

-- 3. 完备度量空间 + 线性结构 = 巴拿赫空间（Banach）
type BanachSpace v = (NormedSpace v, CompleteSpace v)

-- 4. 完备内积空间 = 希尔伯特空间（Hilbert）
type HilbertSpace v = (InnerProductSpace v, CompleteSpace v)
```

### 4. 终极抽象：范畴论 (Category Theory)

```haskell
-- 范畴 (Category)：对象 + 态射 + 组合律
class Category cat where
    type Obj cat a :: Constraint  -- 对象约束
    id :: Obj cat a => cat a a    -- 恒等态射
    (.) :: (Obj cat a, Obj cat b, Obj cat c) 
        => cat b c -> cat a b -> cat a c  -- 组合
    -- 定律: 
    --   id . f = f
    --   f . id = f
    --   (f . g) . h = f . (g . h)

-- Haskell 类型和函数构成一个范畴（Hask）
instance Category (->) where
    type Obj (->) a = ()  -- 所有 Haskell 类型都是对象
    id = \x -> x
    f . g = \x -> f (g x)
```

### 5. 如何用这套词典“阅读”数学概念

当你遇到一个新数学结构时，按以下流程：

| 步骤 | 操作 | Haskell 表达 |
|:---|:---|:---|
| 1. 看它有什么元素 | 确定类型 `a` | `data MyStructure = ...` |
| 2. 看它有什么运算 | 定义函数签名 | `op :: a -> a -> a` |
| 3. 看运算满足什么规律 | 声明 Typeclass | `class MyStructure a where` |
| 4. 看它继承了什么结构 | 添加父类约束 | `class (MetricSpace a, Group a) => MyStructure a` |
| 5. 看能否用程序证明 | 编写 QuickCheck 属性 | `prop_assoc x y z = (x <> y) <> z == x <> (y <> z)` |

### 6. 一个完整示例：L2 希尔伯特空间

```haskell
-- 平方可积函数空间 L2
data L2Func = L2Func (Double -> Double)  -- 实际需处理积分

instance VectorSpace L2Func where
    zero = L2Func (const 0)
    (L2Func f) ^+^ (L2Func g) = L2Func (\x -> f x + g x)
    c *^ (L2Func f) = L2Func (\x -> c * f x)

instance InnerProductSpace L2Func where
    dot (L2Func f) (L2Func g) = integrate (\x -> f x * g x)  -- 积分

-- 这是一个希尔伯特空间（如果完备）
instance CompleteSpace L2Func where
    limit seq = -- 极限函数（在 L2 范数下收敛）
        L2Func (\x -> limit (map ( \n -> eval (seqGen seq n) x) [1..]))
  where
    eval (L2Func f) x = f x
```
