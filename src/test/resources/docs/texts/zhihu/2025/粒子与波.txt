先说波动，波动是个事件时空过程。

它是指参与到这个事件时空（也就是有着明确时间与空间起止点的一个标定范围内）的各种成员（结构），全都无一例外地进入到了某种模式循环。换句话说，它们停止了演进与发展，在某种强大的力量介入下，进入到一种周而复始的往复状态。就像是空循环剧一样，男主角反复地经历一个故事。

如果，我们把男主角视为一个质点，那么该质点的故事时间就是相位；他在特定相位中的心理情绪，比如多巴胺分泌水平，就是振幅；故事的时长就是周期；单位时间（比如一天或是一秒）所经历的循环次数就是频率。

我们再来给男主角设计一只手表，表盘的刻度或者说周长标定成2π。则，表的秒针速度就是角速度ω，代表单位时间里划过了多大比例的2π长度。其实，这个循环在物理上被叫做震动。于是，男主角就可以用结构Aexp(iωt)来进行造型设计。

为什么可以如此呢？

因为exp(iπ)= -1（欧拉公式）。

当我们定义i² = -1以后，就会发现序列{iⁿ}会呈现出“长度为4、节律型为[1, i, -1, -i]”的周期性，即：i⁰ = 1，i¹ = i，i² = -1，i³ = -i，i⁴ = 1，……

注意，序列{iⁿ}的节律型[1, i, -1, -i]的虚实分布特征：实虚交替，奇虚偶实！序列从0开始，每个奇数位上是虚数，每个偶数位上是实数！

又由于：
sin(x)的n阶导数是sin(x + nπ/2)；序列{sin(nπ/2)}具有周期性，即sin(0π/2) = 0，sin(1π/2) = 1，sin(2π/2) = 0，sin(3π/2) = -1，sin(4π/2) = 0，……

注意，序列{sin(nπ/2)}的节律型[0, 1, 0, -1]的零值分布特征：所有偶数位置全都是0！

cos(x)的n阶导数是cos(x + nπ/2)；序列{cos(nπ/2)}具有周期性，即cos(0π/2) = 1，cos(1π/2) = 0，cos(2π/2) = -1，cos(3π/2) = 0，cos(4π/2) = 1，……

注意，序列{cos(nπ/2)}的节律型[1, 0, -1, 0]的零值分布特征：所有奇数位置全都是0！

根据泰勒级数的定义f(x) = sum{f⁽ⁿ⁾(0)/n! * xⁿ}：

sin(x) = sum{sin⁽ⁿ⁾(0)/n! * xⁿ} = sum{sin(0 + nπ/2)/n! * xⁿ} = 0 + x/1! + 0 - x³/3! + 0 + x⁵/5! + ……

cos(x) = sum{cos⁽ⁿ⁾(0)/n! * xⁿ} = sum{cos(0 + nπ/2)/n! * xⁿ} = 1 + 0 - x²/2! + 0 + x⁴/4! + 0 + ……

exp(x) = sum{exp⁽ⁿ⁾(0)/n! * xⁿ} = sum{1/n! * xⁿ} = 1 + x/1! + x²/2! + x³/3! + x⁴/4! + x⁵/5! + ……

于是，
exp(i * x) = sum{1/n! * (i * x)ⁿ} = 1 + i * x/1! - x²/2! - i * x³/3! + x⁴/4! + i * x⁵/5! + …… = (1 + 0 - x²/2! + 0 + x⁴/4! + 0 + ……) + i * (0 + x/1! + 0 - x³/3! + 0 + x⁵/5! + ……) = cos(x) + i * sin(x)

于是，exp(iπ) = cos(π) + i * sin(π) = -1 + 0 = -1。

还有，exp(2iπ) = exp(iπ) * exp(iπ) = -1 * -1 = 1。

结果，Aexp(iωt) = Aexp(iωt) * exp(2iπ) = Aexp(iωt + 2iπ) = Aexp(iω(t + 2π/ω))。

即，Aexp(iωt)以2π/ω为周期。这又刚好对应了那只手表的设计！
